# label元素

 - 语法

        <label>文本</label>

 - 主要属性

  - for：表示与该元素相联系的控件的ID值

 - 作用

  - 将文本与控件联系在一起后，单击文本，效果就同单击控件一样

          <input type="checkbox" name="chkHid" id="chkHid" /> 

          <label for="chkHid">不要公开我的信息</label>

# 选项框

 - 两种：下拉选项框和滚动列表

 - select:创建选项框

    - name:选项框命名
    - size:大于1，则为滚动列表
