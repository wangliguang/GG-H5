# 什么是定位

 - 普通流定位

   - 将页面中的块级元素框从上到下一个接一个地排列

   - 每一个块级元素都会出现在一个新行中

   - 内联元素将在一行中从左到右排列水平布置

 - 浮动

 - 相对定位

 - 绝对定位

 - 固定定位


# 定位属性

  ![](/assets/屏幕快照 2016-10-05 下午12.24.58.png)

 - position属性：更改定位模式为相对定位、绝对定位或者固定定位

   - position:static/relative/absolute/fixed

 - 偏移属性：实现元素框位置的偏移

    - top/bottom/right/left:value

 - 堆叠顺序

    - z-index:value
