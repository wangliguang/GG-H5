# 变量的声明

 - 使用关键字var声明变量，如

       var userName;

 - 使用=为变量赋值

       var bookPrice=25.5;

 - 没有初始值的变量自动取值为undefined

       var publishDate;
       console.log(publisDate);

# 一条语句中声明多个变量

 - 可以在一条语句中声明多个变量，变量名使用；分隔

       var name1,name2,name3;
       var age1,age2=30;

# 命名规范

  - 标识命名规范
  
   - 可以包含字母，数字、下划线(_)、美元符号($)

   - 不能以数字开头

   - 常用语表示函数、变量等的名称
   - 名称最好有明确的含义。
   - 采用驼峰命名法
