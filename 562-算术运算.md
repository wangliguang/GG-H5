* # 算术运算符


* 加\(+\)、减\(-\)、乘\(\*\)、除\(\/\)、求余\(%\)

  * -可以表示减号，也可以表示负号，如x=-y
  * +可以表示加法，也可以用于字符串的连接

    ```
    var num1 = 20-5; //15
    var num2 = "20"-"5";//15
    var num3 = "20a"-5;//NaN
    var num4 = "20"-"5a"; //NaN
    ```

# 使用%运算符

* %表示求余运算

  ```
  var i=10%3; //1
  var i=10.5%3;//1.5
  var i=-10%3;//-1
  var i=10%-3;//1
  var i=-10%-3;//-1
  ```

# 使用++和--运算符

    i++;//相当于i=i+1;

    i=1；
    j=i++;//i结果为2，j结果为1
