# 使用=进行赋值运算

 - =用于为变量赋值

       var score = 90;
       var isFullMarks = score == 100;

- **var x=y=z=10;这种写法不推荐使用**

# 使用扩展赋值表达式

 - +=、-=、*=、/=、%=

       var a = 10; 

       a += 3; //相当于 a = a + 3

       a -= 3; //相当于 a = a - 3

       a *= 3; //相当于 a = a * 3

       a /= 3; //相当于 a = a / 3

       a %= 3; //相当于 a = a % 3

 - **在绝大多数编程语言中，a+=3由于运行时可以进行优化，执行效率都要优于a=a+3**
