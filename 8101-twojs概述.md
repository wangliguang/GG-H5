# Two.js是什么

 - Two.js是一个面向现代Web浏览器提供绘制二维图形的API，它允许使用不同的上下文，而使用相同的API进行绘制

- Two.js所支持的上下文

 - SVG
 - Canvas
 - WebGL

- [官方网址](http://jonbr1.github.io/two.js/)

# Two.js的特点

 - 专注于矢量图

   - Two.js是深度实现具有动画效果的矢量图
   - Two.js致力于更简洁地创建矢量图及动画效果
   - Two.js不支持文本或图片

- 场景

 - Two.js的核心是依赖于场景

 - 当创建或绘制一个对象(Two.polygon或Two.Group)时，是同时存储并记忆
