# 数字类型

 - 功能描述：只能接受数字值

 - 语法

       <input type="number" />
![](/assets/屏幕快照 2016-09-04 下午4.21.47.png)

 - 属性

  - min:当前域你能接受的最小值

  - max:当前域能接受的最大值

  - step: 决定了域所接受值递增或递减的步长，默认为1

# 范围类型

 - 功能描述：允许用户选择一个范围的数值

 - 语法

       <input type="range" min="10" max="1000" value="50" />

 - 属性

  - min：范围的下限值
  - max：范围的上限值
  - step:声明该值递增或递减的步长
  - value:设置初始值
