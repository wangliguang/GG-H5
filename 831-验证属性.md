# Required属性

 - Required属性主要防止域为空时提交表单，该属性不需要设置任何值

 - 语法

       <input type="text" required />
![](/assets/屏幕快照 2016-09-05 上午11.05.40.png)

# Pattern属性

 - Pattern属性的作用是实现元素的验证，他支持使用正则表达式定制验证规则

 - 语法

       <input type="text" pattern="13[0-9]\d{8}" />
